# 🍽️ Swiggy Sales Analysis | Python Data Analytics Project

## 📌 Project Overview :

This project analyzes Swiggy food delivery sales data to uncover insights into revenue trends, customer ordering behavior, and regional performance. The objective was to perform exploratory data analysis (EDA), build key performance indicators (KPIs), and identify sales growth patterns across different cities, states, and food categories.


## 🎯 Business Objectives

* Monitor monthly and daily sales trends
* Identify top-performing cities and states
* Analyze sales distribution by food category
* Track quarterly performance consistency
* Build KPIs to measure overall business performance

## 🛠️ Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib / Seaborn
* Jupyter Notebook

## 📊 Key Analysis Performed

✔ Data Cleaning & Quality Checks
✔ Metadata & Data Type Validation
✔ KPI Development (Total Revenue, Orders, Average Rating)
✔ Monthly & Daily Sales Trend Analysis
✔ Sales by Food Category
✔ State-wise Revenue Distribution
✔ Quarterly Performance Summary
✔ Top 5 Cities by Revenue


## 📈 Key Insights

* Identified peak sales periods through monthly and daily trend analysis
* Detected high-performing cities contributing majority of total revenue
* Analyzed seasonal sales fluctuations using quarterly summaries
* Evaluated category-wise demand to understand customer preferences
